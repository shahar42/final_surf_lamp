# Supabase MCP Server Tools

- `get_database_schema`: Shows all database table structures.
- `query_table`: Queries a specific database table.
- `execute_safe_query`: Runs complex custom SELECT queries.
- `get_table_stats`: Shows row counts and columns.
- `check_database_health`: Verifies database connection is healthy.
- `get_user_dashboard_data`: Fetches all data for user.
- `get_surf_conditions_by_location`: Gets surf conditions by location.
- `get_lamp_status_summary`: Summarizes status of all lamps.
- `search_users_and_locations`: Finds users by name, location.
- `insert_record`: Adds a new database record.
- `delete_record`: Deletes a specific database record.
